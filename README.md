# 💧 SANMUL (샌물) 브랜드 사이트

프리미엄 천연 생수 브랜드 **'샌물'**(가상 생수 브랜드)의 정체성을 담은 인터랙티브 웹사이트입니다.
애플의 예전 제품 브랜드 사이트를 참고하여 만들었습니다. (지금 애플 제품 브랜드 사이트와 다를 수 있습니다.)

## 🔗 배포 주소
- **URL:** [https://sanmul.vercel.app](https://sanmul.vercel.app)

## 🛠 사용 기술
- **Language:** Html / JavaScript & TypeScript
- **Bundler:** Webpack
- **Transpiler:** Babel
- **Styling:** SCSS, CSS
- **Animation:** Canvas API (스크롤 값에 맞춰 전환되는 스크롤 애니메이션)
- **Deployment:** Vercel

## 🚀 실행 방법
1. 저장소 클론
   ```bash
   git clone [https://github.com/아이디/저장소명.git](https://github.com/아이디/저장소명.git)

2. 패키지 설치
   npm install

3. 로컬 실행
   npm start

## 🎥 주요 기능
  - 600장의 프레임 이미지를 활용한 부드러운 스크롤 애니메이션 구현
  - storke 을 긋는 듯한 애니메이션
  - 스크롤 값에 따른 스크롤 애니메이션
  - 반응형 모바일 사이즈 대응
  - Webpack 설정을 통한 자산 최적화 및 빌드 자동화
  - Babel과 Polyfill을 활용한 구형 브라우저 호환성 확보
